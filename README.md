## 创建React项目
npx create-react-app app-react-demo<br/>
cd app-react-demo<br/>
npm run start / npm start<br/>
注意：运行环境 node>=8.1.0

## 环境介绍
node_nodules：包文件<br/>
public：入口文件<br/>
src：源码文件夹
